Types Provider

Similar to the Expression Plugin, this plugin adds support for easily exposing new DataTypes, formats, and validations to Player’s content.

The swift TypesProviderPlugin enables adding custom data types, formatters and validation purely through swift code. While in general, the recommendation would be to share a single JavaScript implementation to multiple platforms, some use cases may need a native integration.

CocoaPods

Add the subspec to your Podfile.

In your Swift file, import the pod. Only one import is needed for all PlayerUI pods.

Swift Package Manager

Add the product to the appropriate target's dependencies in your Package.swift.

In your Swift file, import the sub-package. A different import is needed for each PlayerUI sub-package.
